# Екосистема Monobank

## Overview

Monobank — один із найпопулярніших мобільних банків в Україні, відомий зручним інтерфейсом, кешбеком та інноваційними функціями для управління грошима. Запущений у 2017 році, він швидко зріс завдяки повністю цифровому досвіду — без фізичних відділень і паперової тяганини — з функціями, побудованими навколо того, як люди реально управляють фінансами через телефон. Для користувачів, які хочуть краще управляти фінансами, Monobank пропонує вбудовані інструменти: автоматичну категоризацію транзакцій, статистику витрат, ощадні Банки та мультивалютні рахунки, що замінюють окремі додатки для бюджетування. Підхід банку — зробити фінансові дані видимими та дієвими, показуючи розбивку за категоріями, кешбек та прогрес заощаджень прямо у додатку — встановив стандарт українського цифрового банкінгу. Розуміння цих інструментів допомагає максимально використовувати можливості, які вже є у вашому банківському додатку, без необхідності експортувати дані чи вести зовнішні таблиці.

## Key Concepts

- **Банки (Jars)**: Цифрові «банки» для заощаджень із назвою та ціллю. Можна створити окремі банки для подушки безпеки, відпустки, техніки тощо. Гроші в Банках відокремлені від основного балансу, що зменшує спокусу витратити їх.
- **Рахунки**: Monobank дозволяє мати кілька рахунків (UAH, USD, EUR). Рахунки можна використовувати для розподілу коштів за призначенням — повсякденні витрати, оренда, заощадження.
- **Кешбек**: Monobank повертає відсоток від покупок у визначених категоріях (зазвичай 1-5%). Категорії кешбеку змінюються щомісяця. Це безкоштовні гроші — за умови, що ви не витрачаєте більше заради кешбеку.
- **Автоматична категоризація**: Кожна транзакція автоматично отримує категорію (продукти, транспорт, розваги тощо) на основі коду торговця (MCC). Це створює готову статистику витрат без ручного введення.
- **Виписка та статистика**: Monobank надає візуальну статистику витрат за категоріями, місяцями та періодами. Це ключовий інструмент для розуміння закономірностей власних витрат.

## Practical Examples

### Example: Система Банок для фінансових цілей

Користувач створює три Банки в Monobank:
- «Подушка безпеки» — ціль ₴90 000, автоматичне поповнення ₴5 000/місяць
- «Літня відпустка» — ціль ₴20 000, ручне поповнення з кожного кешбеку
- «Новий iPhone» — ціль ₴45 000, ₴2 000/місяць

Кожна Банка показує прогрес до цілі. Гроші ізольовані від повсякденного балансу, що знижує ризик витратити заощадження на імпульсивні покупки.

### Example: Використання категоризації для бюджету

Людина відкриває статистику в Monobank за місяць і бачить:
- Продукти: ₴5 200
- Кафе та ресторани: ₴3 800
- Транспорт: ₴2 100
- Розваги: ₴1 900
- Шопінг: ₴4 300

Без жодного ручного відстеження вона має повну картину витрат. Виявляється, кафе + ресторани (₴3 800) — це майже стільки ж, скільки основні продукти. Ця інформація допомагає прийняти усвідомлене рішення.

## Actionable Takeaways

1. Налаштуйте Банки для ваших фінансових цілей — візуальний прогрес та ізоляція від основного балансу суттєво допомагають з дисципліною заощадження. Створіть окремі Банки для подушки безпеки, відпустки та конкретних покупок, щоб кожна ціль мала свій прогрес-бар і захист від щоденних витрат.
2. Перевіряйте статистику витрат щотижня — Monobank збирає та категоризує дані автоматично, але ці дані корисні лише якщо ви їх переглядаєте. Щотижневий 5-хвилинний перегляд виявляє тенденції до того, як вони стануть проблемами.
3. Слідкуйте за щомісячними категоріями кешбеку та плануйте більші покупки на місяці з відповідним кешбеком. Це безкоштовні гроші на витрати, які ви і так зробили б, але уникайте зайвих витрат лише заради кешбеку.
4. Виправляйте неправильно категоризовані транзакції вручну для точнішої статистики. Автокатегоризація Monobank використовує коди торговців (MCC), які зазвичай точні, але деякі покупки (змішані ритейлери, ринкові продавці) можуть потребувати корекції.
5. Використовуйте функцію автоматичного поповнення Банок для систематичного заощадження. Налаштувавши переказ ₴3 000-₴5 000 у день зарплати, ви реалізуєте принцип «заплати собі першому» без жодних зусиль щомісяця.
