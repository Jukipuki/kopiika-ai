# Основи оподаткування в Україні

## Overview

Розуміння базових принципів оподаткування в Україні допомагає краще планувати фінанси та уникати несподіванок — особливо різниці між брутто (нарахованою) та нетто (на руки) зарплатою, яка і є реальною сумою для вашого бюджету. Українська податкова система має кілька основних податків для фізичних осіб: ПДФО (18%), військовий збір (5% станом на 2025-2026), ЄСВ (сплачується роботодавцем) та систему для підприємців (ФОП). Для найманих працівників ці податки утримуються автоматично, тобто брутто-зарплата ₴40 000 перетворюється на приблизно ₴30 800 «на руки» — скорочення на 23%, яке стає несподіванкою для тих, хто вперше бачить свій розрахунковий лист. Для фрілансерів та ФОП розуміння своєї групи оподаткування та термінів сплати є критичним для уникнення штрафів та точного планування грошового потоку. Цей документ — освітній огляд того, як працюють ці податки, а не податкова консультація для конкретних ситуацій.

## Key Concepts

- **ПДФО (податок на доходи фізичних осіб)**: Основний податок на дохід — 18% від заробітної плати та більшості інших доходів. Для найманих працівників утримується роботодавцем автоматично.
- **Військовий збір**: Додатковий збір 5% (станом на 2025-2026) від доходу, який утримується разом з ПДФО.
- **ЄСВ (єдиний соціальний внесок)**: Внесок на соціальне забезпечення — 22% від зарплати, який сплачує роботодавець. Для ФОП — фіксована сума або відсоток залежно від групи.
- **ФОП (фізична особа-підприємець)**: Форма ведення бізнесу для фрілансерів та малого бізнесу. Є кілька груп з різним оподаткуванням: 1-ша група (до 1 118 060 грн/рік, ставка ₴302.80/місяць), 2-га група (до 5 587 800 грн/рік, ставка ₴1 514/місяць), 3-тя група (5% від доходу або 3% з ПДВ).
- **Податкова соціальна пільга**: Зменшення бази оподаткування для працівників з низьким доходом. Застосовується автоматично, якщо зарплата не перевищує визначений поріг.

## Practical Examples

### Example: Розрахунок «на руки» для найманого працівника

Роботодавець нараховує зарплату ₴40 000 (брутто):
- ПДФО (18%): ₴7 200
- Військовий збір (5%): ₴2 000
- **На руки (нетто): ₴30 800**

Додатково роботодавець платить ЄСВ (22%): ₴8 800. Загальна вартість працівника для компанії: ₴48 800.

### Example: Порівняння ФОП 3-ї групи та найманої роботи

Фрілансер заробляє ₴50 000/місяць як ФОП 3-ї групи (5%):
- Єдиний податок (5%): ₴2 500
- ЄСВ (мінімальний): ~₴1 760
- **Чистий дохід: ~₴45 740**

Як найманий працівник з тією ж брутто-зарплатою ₴50 000:
- ПДФО + ВЗ (23%): ₴11 500
- **На руки: ₴38 500**

Різниця суттєва, але ФОП не має оплачуваних відпусток, лікарняних та інших соціальних гарантій.

## Actionable Takeaways

1. Розумійте різницю між брутто та нетто зарплатою — нетто є реальною сумою для бюджетування. Завжди запитуйте про нетто при оцінці вакансій, адже пропозиція ₴40 000 брутто означає лише ₴30 800 на вашому рахунку після ПДФО та військового збору.
2. Якщо ви ФОП — відкладайте суму на податки одразу при отриманні доходу, щоб квартальна сплата не стала несподіванкою. Окрема «податкова Банка» в Monobank або виділений рахунок запобігає випадковому витрачанню грошей, які належать податковій.
3. Перевіряйте, чи маєте право на податкову соціальну пільгу — вона може зменшити суму ПДФО для працівників з невисоким доходом. Пороги та розмір пільги оновлюються щорічно.
4. Зберігайте документи про доходи та витрати — вони можуть знадобитися для податкової декларації, особливо для ФОП. Хороші записи також допомагають перевірити правильність утримань роботодавцем.
5. При зміні статусу з найманого працівника на ФОП порахуйте повну різницю, включаючи втрачені соціальні гарантії (оплачувана відпустка, лікарняні, декрет, внески ЄСВ від роботодавця). Різниця в «на руки» може бути значною, але втрата захисту — теж.
